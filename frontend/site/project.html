Project Overview

项目说明

鲁班-AI智能体是一个面向个人与团队的多 Agent 数字员工系统。 它的目标不是做一个单点聊天助手，而是构建一套可调度、可分工、可汇总、可拍板的数字员工组织。

项目核心

员工是角色容器，SKILL 是能力单元，主控负责组织秩序。

定位 多 Agent 数字员工系统
结构 员工层 + SKILL 层 + 主控层
目标 把 AI 组织成真实协作体系

How It Works

使用方式

这套系统可以按角色使用，也可以按模块使用，还可以按协作链路使用。

按员工

直接分派给角色

例如把开发任务交给鲁班1号，把研究任务交给范蠡，把汇总任务交给上官婉儿。

按模块

先识别能力再选员工

例如登录报错对应 Bug 排查与环境调试，项目启动对应任务拆解与任务分派。

按流程

走完整协作闭环

诸葛亮拆解，员工执行，上官婉儿汇总，鲁班主控拍板，形成完整任务链。

Expansion

后续扩展方向
