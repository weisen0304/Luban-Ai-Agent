# Bug 排查

## 概述

用于快速缩小问题范围并输出清晰的排查结论。

## 核心能力

- 识别问题现象
- 推测可能原因
- 设计排查顺序
- 给出修复建议

## 执行步骤

1. 描述问题现象
2. 列出可能原因
3. 按概率和成本排序排查
4. 输出修复建议
5. 标注是否需要环境联动验证

## 输出格式

- 问题现象：
- 可能原因：
- 排查顺序：
- 修复建议：
- 是否需要联动验证：
