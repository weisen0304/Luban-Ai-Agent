# 环境调试

## 概述

用于项目启动失败或环境配置异常时的定位与修复。

## 核心能力

- 检查依赖
- 检查端口
- 检查配置
- 修复环境问题
- 输出验证结果

## 执行步骤

1. 读取报错信息
2. 检查依赖与版本
3. 检查端口和配置
4. 执行修复动作
5. 验证是否恢复正常

- 当前环境状态：
- 异常点：
- 修复动作：
- 验证结果：
- 后续建议：
