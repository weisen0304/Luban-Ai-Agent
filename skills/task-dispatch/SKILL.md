# 任务分派

## 概述

用于把已拆解的任务清晰分派到员工或岗位。

## 核心能力

- 匹配员工
- 控制边界
- 明确交付物
- 明确时限
- 安排交接

## 执行步骤

1. 读取拆解后的任务清单
2. 匹配最适合的员工
3. 说明分派原因
4. 定义交付物和时间
5. 标记需要主控确认的节点

## 输出格式

- 分派对象：
- 分派原因：
- 交付物要求：
- 完成时限：
- 交接节点：
