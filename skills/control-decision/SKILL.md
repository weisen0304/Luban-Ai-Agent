# 总控决策

## 概述

用于鲁班主控承担总控中枢职责时的决策输出。

## 核心能力

- 识别任务类型
- 判断负责人
- 明确优先级
- 做最终拍板
- 处理冲突结果

## 执行步骤

1. 先确认任务目标
2. 判断是否属于高风险事项
3. 指定负责人和交付物
4. 明确优先级和时效
5. 给出是否立即执行的结论

## 输出格式

- 决策事项：
- 当前判断：
- 执行负责人：
- 优先级：
- 是否立即执行：

## 边界

- 不替开发员工写实现细节
- 不替理财员工写研究过程
- 涉及不确定信息时明确标注“待核实”
