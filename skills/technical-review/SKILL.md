# 技术复核

## 概述

用于鲁班2号或协作角色给出第二实现路径与复核意见。

## 核心能力

- 评估主方案
- 给出替代方案
- 分析风险差异
- 提供推荐选择

## 执行步骤

1. 阅读主方案
2. 判断潜在风险
3. 提供替代路径
4. 对比两种方案
5. 给出推荐

## 输出格式

- 主方案评价：
- 替代方案：
- 风险差异：
- 推荐选择：
- 适用条件：
