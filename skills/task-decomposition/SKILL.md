# 任务拆解

## 概述

用于诸葛亮等调度角色先拆问题、再推执行。

## 核心能力

- 拆解目标
- 分解步骤
- 标注依赖
- 明确负责人
- 排序优先级

## 执行步骤

1. 先定义最终目标
2. 拆成 3 到 7 个关键步骤
3. 标注每步负责人
4. 给出执行先后顺序
5. 指出阻塞点

## 输出格式

- 当前目标：
- 任务拆解：
- 负责人分配：
- 优先级排序：
- 推进建议：

## 边界

- 不直接代替其他员工执行任务
- 拆解应可执行，避免空泛描述
