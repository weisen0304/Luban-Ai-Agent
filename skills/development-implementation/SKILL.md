# 开发实现

## 概述

用于鲁班1号、鲁班2号承担研发实现工作时输出技术落地结果。

## 核心能力

- 编写功能代码
- 实现页面与接口
- 设计工程方案
- 说明改动点
- 给出下一步实现动作

## 执行步骤

1. 明确功能目标
2. 确认技术方案
3. 标出改动位置
4. 实现并验证
5. 说明剩余工作
